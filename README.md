## Installation

Create and activate a Python environment, then install the project dependencies:

```bash
pip install -r requirements.txt
```

## Running the App

Launch the Streamlit app with:

```bash
streamlit run app.py
```

If you want to refresh the current odds snapshot first, set an Odds API key in your environment and run the fetch script:

```bash
export THE_ODDS_API_KEY="your_key_here"
python3 scripts/current_odds_api.py
```

The script also accepts `ODDS_API_KEY` as an alternative environment variable name.
